Internet - Entendendo como ela funciona

Cliente: Firefox, Mozilla, Chrome

Servidor: Servidor é armazenar, processar e entregar páginas web para os Clientes.

Endereço IP: A sigla IP refere-se a Internet Protocol . O endereço IP é um identificador numérico para um dispositivo (computador, celular, tablet, impressora, roteador, servidor...) conectado à rede TCP/IP.

(exemplo: 244.155.65.2). Esses números são chamados de endereço lógico

Esse endereço físico faz parte do seu hardware e é conhecido por endereço de controle de acesso à mídia (MAC address).

TCP/IP: Essa sigla refere-se à Transmission Control Protocol/Internet Protocol .

ISP: ISP ou Internet Service Provider é o seu provedor de Internet. É qualquer empresa que nos fornece acesso ao maravilhoso mundo Web.

DNS: DNS ou Domain Name System é um dispositivo com uma base de dados distribuída que gerencia os nomes de serviços, computadores ou qualquer dispositivo conectado à Internet

Port Number: O número de porta é um número de 16 bits utilizado para identificar uma porta específica no servidor

Host: O Host é qualquer computador conectado à rede, seja como cliente, servidor ou qualquer outro tipo de dispositivo.

HTTP: Hyper-text Transfer Protocol é o protocolo utilizado para a comunicação entre navegadores e servidores na Internet.

URL: O URL é a sigla para Uniform Resource Locators e identifica um recurso web específico.
